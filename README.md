# Interpretable Chirality-Aware Graph Neural Network for Quantitative Structure Activity Relationship Modeling in Drug Discovery 

by - Yunchao (Lance) Liu, Yu Wang, Oanh Vu, Rocco Moretti, Bobby Bodenheimer, Jens Meiler, Tyler Derr

The main objective of this paper is to introduce a new graph neural network model called Molecular-Kernel Graph Neural Network (MolKGNN) for quantitative structure-activity relationship (QSAR) modelling in computer-aided drug discovery.
